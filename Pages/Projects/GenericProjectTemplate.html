The Dungeons Below

Python Game, Cocos2Dpy, Procedural Map Generation

The Dungeons Below is a hack and slash action game with a realtime charecter and a procedural level generation system. I created this game in python for a class on python programing and later expanded it to create more refined Ai Agents and to implement interesting graphical techniques like autotiling for the procedural tilemaps.

Project Details

  • Category: Game Development, Game Design, Graphics,
  • Date: Fall 2019
  • Technologies: Python, Coso2Dpy, JSON, Procedural Generation
